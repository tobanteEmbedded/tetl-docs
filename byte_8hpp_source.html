10 /// \brief etl::byte is a distinct type that implements the concept of byte as
11 /// specified in the C++ language definition.
12 ///
13 /// \details Like char and unsigned char, it can be used to access raw memory
14 /// occupied by other objects, but unlike those types, it is not a character
15 /// type and is not an arithmetic type. A byte is only a collection of bits, and
16 /// the only operators defined for it are the bitwise ones.
17 ///
18 /// https://en.cppreference.com/w/cpp/types/byte
